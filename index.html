About GhostCrypt

🔒 What is GhostCrypt?

GhostCrypt is a client-side file encryption tool that uses military-grade cryptography to protect your files. Everything happens in your browser - no data is ever sent to a server.

🛡️ Security Architecture

  • 5-Layer Protection: Argon2id → AES-256-CBC → ChaCha20 → HMAC-SHA512
  • Double Encryption: AES-256 + ChaCha20 for defense in depth
  • Key Derivation: Argon2id with 64 MB memory, 3 iterations
  • Integrity Check: HMAC-SHA512 prevents tampering
  • Zero-Knowledge: 100% client-side, no server involvement

⚡ Technical Specifications

  • Protocol: GHOST 1
  • Encryption: AES-256-CBC + ChaCha20
  • Authentication: Password or Keyfile (.gkey)
  • File Size: Up to 1 GB supported
  • Overhead: Only 130 bytes per file
  • Browser Support: Chrome 90+, Firefox 88+, Safari 14+

📖 How It Works

  1. Upload: Select any file from your device
  2. Authenticate: Choose password or keyfile protection
  3. Process: File is encrypted/decrypted in your browser
  4. Download: Save the .ghost file (encrypted) or original (decrypted)

⚠️ Important Notes

  • Always use strong passwords (12+ characters recommended)
  • Store keyfiles in a safe location - lost keyfiles = unrecoverable data
  • No password recovery possible - this is zero-knowledge encryption
  • Keep backups of important files before encryption
